### 遊戲運行：
- 主選單
- 遊戲畫面
- 遊戲開始動畫
- 遊戲結束動畫
- 遊戲結束影片

### 角色：
- 移動
- 移動優化(撞牆自動偏移)
- 放炸彈
- 吃道具
- 死亡判定

### 炸彈：
- 放置
- 消失
- 炸掉方塊(且不會炸穿方塊)
- 連鎖爆炸
- 爆炸動畫

### 道具：
- 跑鞋
- 強化藥水
- 炸彈+1
- 道具位置隨機

### 方塊：
- 無法炸掉的方塊
- 可以炸掉的方塊
- 方塊位置隨機
